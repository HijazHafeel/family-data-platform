üìù How to Save to Dialog Router USB

  1. Click a backup button above to download the backup file(s) to your computer's Downloads folder
  2. Access your Dialog router USB drive:
    • Open File Explorer (Windows) or Finder (Mac)
    • Look for your router's shared network drive (usually \\\\ROUTER-IP\\USB or similar)
    • Or access via router admin panel (usually http://192.168.1.1)
  3. Create a backup folder on the USB drive (e.g., "FamilyDataBackups")
  4. Copy the downloaded backup files from your Downloads folder to the USB drive backup folder
  5. Verify the backup by opening the files to ensure they contain your data
üí° Tip: Create regular backups (weekly or monthly) to ensure your data is always safe. Keep multiple backup copies in different locations.
